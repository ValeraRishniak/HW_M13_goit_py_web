Реалізація проекту Django

Для роботи проекта необхідний файл /quotes_toscrape/.env зі змінними оточення. Створіть його з таким вмістом і підставте свої значення.

SECRET_KEY=

DATABASE_NAME=
DATABASE_USER=
DATABASE_PASSWORD=
DATABASE_HOST=
DATABASE_PORT=

EMAIL_HOST=
EMAIL_PORT=
EMAIL_HOST_USER=
EMAIL_HOST_PASSWORD=

Запуск застосунку

cd quotes_toscrape
py manage.py runserver
